Benchmarks

Benchmarks for a few representative systems and compilers are given below. The results include real and complex matrix multiplies in single and double precision.

  • ATLAS 3.10.0, Cortex A9 (TI OMAP 4), dual core with NEON
  • ATLAS 3.8.4 ARM, Cortex A8 (TI OMAP 3), single core with NEON
  • ATLAS 3.8.4 ARM, Cortex A9 (TI OMAP 4), dual core with NEON

If you'd like to duplicate these benchmark tests on your system, you can use our GEMM test program and our GETRF test program to repeat all the computations shown above. (Our results, in the same format, are linked to at the bottom of each benchmark page.) We'd be interested to see any results on CPUs we don't have listed above.
